# Full-stack-development
我的全栈开发的学习笔记

## 项目运行
#### 记得先安装了node.js
```
git clone https://github.com/codeyoyo/Full-stack-development.git
npm install
npm run server //模拟生产环境运行项目
npm run build //打包项目
```
